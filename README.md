# NeverNote

なんでもメモアプリ

# Getting Start

```
$ git clone https://github.com/hito-kotaro/NeverNote.git
$ cd NeverNote
$ yarn
$ yarn start
```

# URL

TopPage: http://localhost:3000/  
LoginPage: http://localhost:3000/login  
HomePage: http://localhost:3000/home  
NotePage: http://localhost:3000/note
